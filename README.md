# Weather Dataset Analysis Using Python

## Project Overview

This project is a beginner-level data analysis and preprocessing project created in Jupyter Notebook using Python.

The notebook focuses on:

* loading and exploring a weather dataset,
* understanding dataset structure,
* handling and preprocessing data,
* and experimenting with beginner machine learning workflow using Python libraries.

This project was created as part of practical learning in Python-based data analysis.
